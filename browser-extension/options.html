API Configuration

Connect your Anthropic API key to enable AI-powered translations. Without an API key, the extension works in demo mode with limited functionality.

Demo Mode - Add API key for full functionality

How to get an API key

  1. Go to console.anthropic.com
  2. Create an account or sign in
  3. Navigate to API Keys section
  4. Create a new key and copy it here

Usage

Three ways to translate legal text:

  • Click the extension icon and paste text directly
  • Select text on any page and it auto-fills when you open the popup
  • Right-click selected text and choose "Translate with PlainSpeak"
